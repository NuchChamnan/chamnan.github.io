User Guideline

Mozilla Thunderbird

  1. Open Thunderbird and go to Account Settings from the menu or by right-clicking your account.
  2. Click "Manage Identities" button.
  3. In the identity window:
    • Click "Add"
    • Set email to demo@your-name.com
  4. Click "OK" and restart Thunderbird.
Note: Thunderbird may take a few minutes to fully apply changes.

Microsoft Outlook

  1. Go to File > Account Settings.
  2. Select your account, then click "Change".
  3. In the new window:
    • Click "More Settings"
    • Open "Send As" tab
    • Set email to demo@your-name.com
  4. Click OK → Next → Finish. Restart Outlook.
Important: You may need administrator access for these changes.

Google Mail (Gmail)

  1. Go to Settings → See all settings → Accounts and Import.
  2. Click "Add another email address".
  3. Fill out the popup:
    • Name + demo@your-name.com
    • Uncheck "Treat as alias"
    • Click "Next Step"
  4. Complete verification:
    • Check email for code
    • Enter it and click Verify
  5. Now you can select the new address when sending emails.
Tip: Verification usually takes 1–2 minutes. Check spam if needed.
